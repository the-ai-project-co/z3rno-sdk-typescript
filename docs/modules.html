      @z3rno/sdk

      Z3rno TypeScript SDK — AI agent memory database client.

      This is the main entry point for the @z3rno/sdk package. It re-exports the Z3rnoClient class, all error types, and all Zod schemas / TypeScript types used by the API.

      Example

      import { Z3rnoClient } from "@z3rno/sdk";

      const client = new Z3rnoClient({
        baseUrl: "http://localhost:8000",
        apiKey: "z3rno_sk_..."
      });

      const memory = await client.store({
        agentId: "agent-1",
        content: "User prefers dark mode",
      });
